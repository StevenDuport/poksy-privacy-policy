Politique de Confidentialité - Poksy

Dernière mise à jour : 30 octobre 2025

1. Introduction

Bienvenue sur Poksy, votre application de gestion budgétaire personnelle. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations personnelles.

En utilisant Poksy, vous acceptez les pratiques décrites dans cette politique de confidentialité.

2. Informations que nous collectons

2.1 Données personnelles et de compte

Lors de la création de votre compte, nous collectons :

  • Adresse email (via Firebase Authentication)
  • Identifiant utilisateur unique (UID)
  • Informations de profil (nom, dates de salaire, préférences)

2.2 Données budgétaires stockées dans le cloud

Toutes vos données financières sont stockées de manière sécurisée dans Firebase Firestore (Google Cloud) :

  • Transactions : Vos revenus, dépenses, factures et épargnes
  • Budgets mensuels : Vos budgets prévisionnels et réels
  • Catégories : Vos catégories personnalisées de dépenses
  • Profil utilisateur : Soldes, dates de salaire, préférences

Sécurité : Ces données sont :

  • ✅ Stockées de manière chiffrée sur les serveurs Google Cloud
  • ✅ Accessibles uniquement par vous via votre compte
  • ✅ Protégées par des règles de sécurité Firebase strictes
  • ✅ Synchronisées entre vos appareils si vous utilisez le même compte

2.3 Informations d'abonnement

Si vous souscrivez à un abonnement Premium, nous collectons et stockons :

  • Informations de transaction via RevenueCat (identifiant anonyme)
  • Statut d'abonnement (actif, expiré, période d'essai)
  • Date d'expiration et renouvellement automatique
  • Plateforme d'achat (App Store)

2.4 Données d'utilisation et techniques

Nous pouvons collecter des informations anonymes sur l'utilisation de l'application :

  • Type d'appareil et version du système d'exploitation
  • Données de crash et de performance (via Expo)
  • Fonctionnalités utilisées (de manière agrégée et anonyme)
  • Logs d'erreurs pour améliorer la stabilité

3. Comment nous utilisons vos informations

Nous utilisons vos informations pour :

  • Fournir et améliorer les fonctionnalités de l'application
  • Gérer votre abonnement Premium
  • Assurer le support technique
  • Améliorer la sécurité et la stabilité de l'application
  • Respecter nos obligations légales

4. Services tiers

4.1 Firebase / Google Cloud (Firestore & Authentication)

Nous utilisons Firebase de Google pour :

  • Firebase Authentication : Gestion sécurisée de votre connexion et authentification
  • Firebase Firestore : Stockage cloud sécurisé de vos données budgétaires

Toutes vos données sont hébergées sur les serveurs Google Cloud et soumises aux politiques de confidentialité de Firebase et aux politiques de confidentialité de Google.

Localisation des données : Les données sont stockées dans des centres de données Google Cloud certifiés, conformes au RGPD.

4.2 RevenueCat

Nous utilisons RevenueCat pour gérer les abonnements in-app. RevenueCat collecte et traite des informations liées à vos achats conformément à leur politique de confidentialité.

4.3 Apple App Store

Les achats sont traités par Apple. Apple collecte et traite vos informations de paiement conformément à leur politique de confidentialité. Nous n'avons pas accès à vos informations de carte bancaire.

4.4 Expo

L'application est développée avec Expo. Des données de performance anonymes peuvent être collectées pour améliorer la stabilité de l'application.

5. Partage des données

Nous ne vendons jamais vos données personnelles. Nous ne partageons vos informations qu'avec :

  • Firebase/Google Cloud : Pour le stockage et la synchronisation de vos données
  • RevenueCat : Pour la gestion des abonnements Premium
  • Apple : Pour le traitement des paiements in-app
  • Autorités légales : Si requis par la loi

Important : Nous ne partageons jamais vos données avec des tiers à des fins marketing ou publicitaires.

6. Sécurité des données

Nous prenons la sécurité de vos données au sérieux et mettons en œuvre plusieurs mesures de protection :

  • Chiffrement en transit : Toutes les communications entre l'app et Firebase sont chiffrées via HTTPS/TLS
  • Chiffrement au repos : Vos données sont chiffrées dans les bases de données Firebase
  • Authentification sécurisée : Firebase Authentication avec protocoles standards (OAuth 2.0)
  • Règles de sécurité : Accès strictement limité à votre compte uniquement
  • Infrastructure certifiée : Google Cloud Platform (ISO 27001, SOC 2, conforme RGPD)
  • Aucun accès administrateur : Nous n'avons pas accès à vos données financières personnelles

En cas de violation de données : Nous vous informerons dans les 72 heures conformément au RGPD.

7. Vos droits

Conformément au RGPD et aux lois sur la protection des données, vous avez le droit de :

  • Accéder à vos données personnelles
  • Rectifier vos données inexactes
  • Supprimer vos données (droit à l'oubli)
  • Vous opposer au traitement de vos données
  • Retirer votre consentement à tout moment

Pour supprimer toutes vos données locales, vous pouvez simplement désinstaller l'application. Pour supprimer vos données d'abonnement, contactez-nous.

8. Conservation des données

  • Données locales : Conservées jusqu'à la désinstallation de l'app
  • Données d'abonnement : Conservées pendant la durée de l'abonnement + 1 an (obligations légales)
  • Données de support : Conservées pendant 3 ans maximum

9. Confidentialité des mineurs

Poksy est destiné aux personnes âgées de 13 ans et plus. Nous ne collectons pas sciemment d'informations personnelles auprès d'enfants de moins de 13 ans.

10. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre. Nous vous informerons de tout changement important via l'application ou par email.
